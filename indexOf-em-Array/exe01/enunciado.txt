Exercício: Encontrando Elementos em um Array
Objetivo
Dado um array de números e um número específico, você deve determinar se o número está presente no array e, em caso afirmativo, retornar o índice da primeira ocorrência desse número.

Descrição do Problema
Crie uma função chamada encontrarIndice que aceita dois parâmetros:

Um array de números (numeros).
Um número específico (numeroProcurado).
A função deve retornar o índice da primeira ocorrência do numeroProcurado no array numeros. Se o número não estiver presente no array, a função deve retornar -1.

Teste a função com diferentes arrays e números.

Dicas
Use o método indexOf() para encontrar a posição do número no array.
Certifique-se de que sua função lida corretamente com números que não estão presentes no array.
